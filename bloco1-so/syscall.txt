# PARTE C — Questões de síntese

# 1. A apresentação afirma que "instruções de I/O são privilegiadas".
#    Como um programa de usuário consegue fazer I/O sem executar
#    instruções privilegiadas diretamente? (mínimo 2 frases)

Resposta 1:

o programa pede pro sistema operacional fazer o i/o por ele, usando chamadas como write()


# 2. Compare write(fd, buffer, tamanho) com o "programa de canal"
#    da apresentação: o que os dois têm em comum em termos de
#    responsabilidades?

Resposta 2:

os dois servem para enviar dados para um dispositivo, o programa passa as informaçoes, e o sistema faz a comunicaçao com o hardware
